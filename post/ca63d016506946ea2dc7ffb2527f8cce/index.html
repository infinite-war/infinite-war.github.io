# 常见Dos命令

CMD(Powershell也可)

# 打开方式

1.开始+系统++命令提示符

2.win+R 输入cmd控制台

3.在任意文件夹下面，按住shift+鼠标右键点击

4.资源管理器的地址栏前加上cmd路径

按右键可以选择以管理员方式运行

# 常用命令

下面忽略"C:\xxx\xxx>"的部分

# 1.盘符切换

F:     //切换至F盘

# 2.查看当前目录下的所有文件

dir

# 3.切换目录

cd 文件路径

可以是绝对路径，也可以是相对路径。

相对路径中：./ 代表当前目录，**../**代表当前目录的上一级目录

# 4.清理屏幕

cls

# 5.退出终端

exit

# 6.网络相关

# 6.1查看电脑的ip

ipconfig

# 6.2ping命令

ping  [各种可用参数]  网页地址

ping命令可以获取指定网址的ip信息

ping 还有其他可用参数，具体可以通过直接输入ping查看

# 7.打开应用

calc  				//计算器
mspaint				//画板
notepad				//文本编辑器

# 8.文件操作

# 8.1 md创建目录

注：这个命令只是创建目录(即平时所说的文件夹)而不是 .txt 这类文件。

md  目录名        //make directory

# 8.2rd删除目录

rd 目录名     //remove directory

# 8.3打开文件

相对路径或绝对路径+文件名

# 8.4创建文件

cd.> 文件名.文件类型    //文件会创建在当前目录

# 8.5删除文件

del 文件名.文件类型

文件

cd.> 文件名.文件类型    //文件会创建在当前目录

# 8.5删除文件

del 文件名.文件类型
